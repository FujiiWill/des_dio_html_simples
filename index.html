Introdução:

Voltar

HTML (HyperText Markup Language) é um conjunto de marcações usadas para criar e estruturar páginas da web. As tags HTML são comandos especiais que dizem ao navegador como exibir o conteúdo da página. Vamos explicar algumas das principais tags HTML e para que elas servem, usando uma linguagem simples e exemplos práticos.

Tags de estruturação: Algumas tags possuem um carater mais estrutural do documento, como veremos a seguir:

Voltar

Tag de lista ordenada e lista não ordenada: '<ol>' e '<ul>'

Voltar

As tags '<ol>' e '<ul>' são usadas para criar listas. A tag '<ol>' cria uma lista ordenada (numerada), enquanto a tag '<ul>' cria uma lista não ordenada (com marcadores). As tags '<li>' são usadas para definir cada item da lista.

Sintaxe da lista ordenada:

Resultado do código da sintaxe:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Sintaxe da lista não ordenada:

Resultado do código da sintaxe:

  • Item um
  • Item dois
  • Item três
